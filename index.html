Daedalus - Minecraft Mini-Game

Daedalus is a Minecraft mini game project. 5 teams arranged in a labyrinth will have to face each other in a bloody fight in order to take control. This plugin has different generation algorithms such as a maze generator, a random structure generator and a loot table generator. Some structure is guarded by a boss with custom attacks! The plugin also has a team and episode event management system with game settings configurator.
